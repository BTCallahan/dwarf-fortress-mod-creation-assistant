About:

This is a tool to make the creation of Dwarf Fortress mods easier and more user frendly. Rather then copying and pasting tokens, the user can use checkboxs, drop-down menus, and text and numberical fields to choose what tokens are used in the mod.

How to use:

Click on the section in the header to navigate to the creation page.

Next, you will probably want to define the number of spaces in each tab. Dwarf Fortress text files use tabs to make the data more readable. If you wan't your mod files to use tab characters instead of spaces, then enter a value of 0.

After you have entered the variables you want, click the button "Create new creture/item/entity/ect" at the bottom of the page. This will generate text with the mod data.
